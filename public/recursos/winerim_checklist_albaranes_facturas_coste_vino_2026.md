# Checklist de albaranes, facturas y coste de vino

Recurso operativo para evitar que tarifas, albaranes y facturas rompan el margen de la carta de vinos por desorden documental, costes antiguos o PVP no revisados.

## Objetivo

Conseguir que cada documento de compra termine en una decisión clara:

- actualizar coste;
- revisar PVP;
- ajustar margen;
- reclamar diferencia;
- bloquear reposición;
- renegociar con proveedor.

## 1. Checklist de recepción

| Control | OK | Incidencia | Acción |
|---|---|---|---|
| El proveedor coincide con el pedido. |  |  |  |
| Las referencias entregadas coinciden con la carta. |  |  |  |
| Formato, capacidad y añada son correctos. |  |  |  |
| El coste unitario coincide con tarifa acordada. |  |  |  |
| Descuentos, portes y rappels están reflejados. |  |  |  |
| IVA y condiciones se tratan de forma consistente. |  |  |  |
| El stock se actualiza tras recepción. |  |  |  |
| El coste de compra se actualiza en la referencia. |  |  |  |

## 2. Umbrales de alerta

| Señal | Umbral recomendado | Acción |
|---|---:|---|
| Subida de coste frente a tarifa anterior | > 3% | Revisar PVP o margen |
| Diferencia entre pedido y factura | cualquier diferencia | Reclamar o registrar incidencia |
| Referencia con coste nuevo y PVP antiguo | inmediato | Bloquear reposición hasta revisar |
| Vino con más de 90 días sin venta | > 90 días | No reponer salvo motivo estratégico |
| Proveedor con incidencias repetidas | 2 incidencias/mes | Auditar condiciones |

## 3. Ruta semanal de control

### Lunes: documentos

- Centralizar albaranes y facturas recibidas.
- Marcar documentos pendientes de validar.
- Separar incidencias por proveedor.

### Martes: coste

- Actualizar coste unitario de referencias afectadas.
- Identificar subidas relevantes.
- Comparar coste actual vs coste usado para fijar PVP.

### Miércoles: margen

- Recalcular margen por referencia impactada.
- Marcar vinos por debajo del margen objetivo.
- Revisar especialmente vinos por copa y top ventas.

### Jueves: compras

- Bloquear reposición de referencias lentas o con margen deteriorado.
- Preparar argumentos de renegociación.
- Buscar alternativas equivalentes si el coste no se sostiene.

### Viernes: decisión

- Actualizar PVP si procede.
- Registrar incidencias cerradas.
- Crear lista de acciones para sala, compras o dirección.

## 4. Preguntas para proveedor

- ¿La subida es puntual o permanente?
- ¿Existe descuento por volumen o por familia de producto?
- ¿Hay alternativa equivalente con mejor coste?
- ¿Se puede mantener precio anterior por rotación o compromiso?
- ¿Qué referencias del catálogo cubren un hueco real de mi carta?
- ¿Qué referencias solapan con otros distribuidores?

## 5. Señales de que necesitas automatizar

- Los documentos llegan por email, WhatsApp, papel y carpetas separadas.
- El PVP se revisa solo cuando alguien se acuerda.
- Compras no ve el impacto real sobre margen.
- La carta muestra precios que no responden al coste actual.
- Nadie sabe cuánto capital hay inmovilizado por proveedor.
