# Perfil RIM del restaurante

Diagnóstico práctico para identificar qué tipo de carta tiene tu restaurante y qué decisiones debería priorizar Winerim: margen, rotación, profundidad, experiencia, simplificación, compras o formación de sala.

## Cómo usarlo

Puntúa cada afirmación de 1 a 5.

- 1 = no ocurre casi nunca.
- 3 = ocurre a medias o de forma irregular.
- 5 = describe muy bien tu carta actual.

Al final, suma cada bloque. El bloque con más puntos marca tu perfil dominante. Si dos perfiles quedan muy cerca, tu carta es híbrida y conviene priorizar el cuello de botella más urgente: margen, stock, venta de sala o claridad para el cliente.

## Bloque A: Carta gourmet

| Señal | Puntuación 1-5 |
|---|---:|
| La carta tiene una identidad clara y coherente con la cocina. |  |
| Hay referencias elegidas por relato, origen, productor o maridaje, no solo por precio. |  |
| El equipo puede explicar los vinos clave con seguridad. |  |
| El ticket medio permite trabajar referencias de mayor valor. |  |
| La carta aporta experiencia, no solo disponibilidad. |  |

Lectura: si este bloque domina, prioriza narrativa, maridajes, formación de sala, profundidad equilibrada y venta consultiva. No midas la carta solo por rotación: algunos vinos tienen valor de imagen, pero deben estar justificados.

## Bloque B: Carta coleccionista

| Señal | Puntuación 1-5 |
|---|---:|
| La carta es amplia y tiene muchas referencias especiales o de guarda. |  |
| Hay capital relevante inmovilizado en bodega. |  |
| Algunas referencias llevan más de 120 días sin venta. |  |
| La carta impresiona, pero cuesta saber qué mover cada mes. |  |
| Comprar bien es tan importante como vender bien. |  |

Lectura: si este bloque domina, prioriza stock, capital inmovilizado, cobertura de meses, señales de no reposición y control por añada/formato. El riesgo no es tener una gran carta: el riesgo es no saber qué parte de la carta está parada.

## Bloque C: Carta estratégica

| Señal | Puntuación 1-5 |
|---|---:|
| Revisas PVP, coste, margen y rotación con cierta frecuencia. |  |
| Te interesa saber qué vinos sostienen margen, no solo ventas. |  |
| Comparas proveedores y alternativas antes de reponer. |  |
| Tienes una oferta por copa pensada para vender y aprender. |  |
| Te gustaría automatizar decisiones de compra y retirada. |  |

Lectura: si este bloque domina, prioriza automatización, benchmarks, alertas de margen, auditoría de distribuidores y scorecards mensuales. Este perfil suele capturar valor rápido con Winerim porque ya existe disciplina de gestión.

## Bloque D: Carta turística

| Señal | Puntuación 1-5 |
|---|---:|
| El restaurante depende de temporadas, picos o clientes de paso. |  |
| La carta necesita ser clara y fácil de decidir. |  |
| Las referencias tractoras deben estar siempre disponibles. |  |
| La oferta por copa tiene peso comercial. |  |
| El equipo necesita recomendaciones simples y rápidas. |  |

Lectura: si este bloque domina, prioriza claridad, vino por copa, roturas de stock, traducciones, filtros, recomendaciones y plan de temporada. El objetivo es reducir fricción y proteger ventas en momentos de alta demanda.

## Bloque E: Carta pasiva

| Señal | Puntuación 1-5 |
|---|---:|
| La carta existe, pero no se vende activamente. |  |
| El equipo recomienda pocas referencias. |  |
| Hay vinos que nadie sabe explicar bien. |  |
| Se repone por costumbre más que por rendimiento. |  |
| El cliente acaba eligiendo por precio o por lo conocido. |  |

Lectura: si este bloque domina, prioriza simplificación, formación exprés, selección de vinos recomendados, copa rentable y reducción de referencias sin papel claro. Aquí el primer objetivo no es sofisticar la carta, sino hacer que venda.

## Bloque F: Carta superviviente

| Señal | Puntuación 1-5 |
|---|---:|
| No hay una rutina clara para revisar carta, stock y compras. |  |
| Los costes cambian pero los PVP tardan en actualizarse. |  |
| Los documentos de compra llegan por varios canales. |  |
| No sabes con seguridad qué vinos dejan margen real. |  |
| La carta depende demasiado de memoria, intuición o urgencias. |  |

Lectura: si este bloque domina, prioriza orden operativo: costes, albaranes, stock, PVP, TPV y revisiones mensuales. No intentes resolver todo a la vez. Empieza por proteger margen y evitar compras a ciegas.

## Resultado rápido

| Perfil dominante | Qué medir primero | Acción recomendada |
|---|---|---|
| Gourmet | Ticket vino, maridajes, referencias estratégicas, capacidad de sala | Reforzar narrativa y recomendaciones |
| Coleccionista | Capital inmovilizado, días sin venta, cobertura, añadas | Detectar no reposición y liquidación selectiva |
| Estratégica | Margen, rotación, proveedores, PVP, copa | Automatizar scorecard y decisiones |
| Turística | Roturas, copa, claridad, temporada, traducción | Preparar carta y equipo antes del pico |
| Pasiva | Recomendaciones, referencias sin papel, venta por copa | Simplificar y formar al equipo |
| Superviviente | Coste, PVP, albaranes, stock, rutina mensual | Ordenar datos mínimos y proteger margen |

## Qué haría Winerim con este diagnóstico

Winerim puede adaptar el análisis según el perfil:

- En una carta coleccionista, vigilaría stock muerto, capital inmovilizado y cobertura.
- En una carta turística, priorizaría roturas, referencias tractoras y venta por copa.
- En una carta estratégica, cruzaría márgenes, TPV, proveedores y reposición.
- En una carta pasiva, marcaría vinos recomendables para sala y simplificación de carta.

## Próxima decisión

Elige una sola prioridad para los próximos 30 días:

1. Proteger margen.
2. Reducir stock parado.
3. Mejorar venta por copa.
4. Ordenar proveedores.
5. Formar sala.
6. Simplificar carta.
7. Preparar temporada.
8. Revisar PVP.

La mejor carta no es la más larga ni la más sofisticada. Es la que encaja con el restaurante, se puede vender, se puede gestionar y se puede mejorar con datos.
